Open this page in Metamask Android embedded browser

  1. First click to set Same-Site Cookie
  2. Then verify 
    SameSite=Strict
    cookie policy is applied

If the browser is applying the SameSite Cookies strict policy, you won't find the 

Super=Secret
cookie in httpbin request header on step 2

Learn more on this on Github issue "Same site" cookie attribute not working on Android browser #252
